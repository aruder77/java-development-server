Allgemeines

  • Der aktuelle Benutzer ist "smoketest" mit Passwort "smoketest"
  • Mit "sudo bash" und Eingabe des smoketest Passworts sind root-Operationen möglich
  • Euer Windows-Verzeichnis X:/shared ist nach /shared gemappt (Vollzugriff)

Oracle

  • Auf die lokale Oracle-DB kann mit folgenden Parametern zugegriffen werden:
    • Hostname: localhost
    • Port: 1521
    • SID: XE
    • Benutzer: DEV01 oder DEV02
    • Passwort: DEV01 oder DEV02
  • sysdba-Zugriff auf die Oracle-Instanz als smoketest-User: sqlplus / as sysdba
  • Der Sqldeveloper kann über den Link auf dem Desktop gestartet werden.

VNC-Zugriff

  • Über VNC kann eine Remote-Desktopverbindung hergestellt werden. Dazu ist ein VNC-Viewer wie z.B. TightVNC erforderlich.
  • Der VNC-Port ist 5900 und wird von der VM exportiert.
